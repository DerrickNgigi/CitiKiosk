Banner Content Change Options

  • Find the banner sections, The Banner sections is well commenting, so that you find it easily.
  • Here you can see the banner text with in the h1 and p tag.
  • You can also see that the banner button, you can also change button text
  • If you need you can change banner content as your need

Banner image Change

  • You find the index.html and go open it in a code editor.
  • Find the banner sections, The Banner sections is well commenting, so that you find it easily.
